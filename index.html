'What Should I Roll?' is a website designed by TeyStar to quickly give you a class for your next character, built around what the current party already contains and the campaign setting.

'What Should I Roll?' is not 100% accurate but will almost always give a good class suggestion. Its accuracy diminishes when players use 3rd party materials or class archtypes. Think about these variables and try to find an archtype within the suggested class for best results in these circumstances. Always try to make something that will be fun to play!
