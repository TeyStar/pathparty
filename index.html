What Should I Roll?
'What Should I Roll?' is a website designed by TeyStar to quickly give you a class idea for your next character, built around what the current party already contains.

'What Should I Roll?' is not 100% accurate but will almost always give a good class suggestion. Do not use it to create an entire party without expecting some wonky results. Its accuracy diminishes when players use 3rd party materials or class archetypes. Think about these variables and try to find an archetype within the suggested class for best results in these circumstances. Always try to make something that will be fun to play!
